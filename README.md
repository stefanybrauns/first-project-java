# First Project Java ☕

Este repositório contém minha primeira aplicação desenvolvida em Java, com foco nos fundamentos da linguagem e na organização de código voltada para backend.

O projeto faz parte do meu início de jornada em desenvolvimento e também da minha transição para a área de Dados, buscando entender como informações são geradas, processadas e estruturadas dentro de aplicações antes de serem analisadas.

---

## 📋 Sobre o Projeto

Esta aplicação foi desenvolvida no **IntelliJ IDEA** com o objetivo de praticar conceitos fundamentais de Java, como:

- Estrutura básica de classes e objetos  
- Entrada e saída de dados  
- Lógica de programação  
- Organização de código  
- Primeiros contatos com desenvolvimento backend  

Este projeto representa minha base inicial de aprendizado na linguagem Java.

---

## 🚀 Como Executar

Para rodar este projeto localmente:

1. **Clone o repositório**
```bash
git clone https://github.com/stefanybrauns/first-project-java.git
```

2. **Acesse a pasta do projeto**
```bash
cd first-project-java
```

3. **Navegue até a pasta `src`**
```bash
cd src
```

4. **Compile o arquivo principal**
```bash
javac Main.java
```

5. **Execute o projeto**
```bash
java Main
```
